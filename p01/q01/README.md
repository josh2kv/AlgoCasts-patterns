# Introduction

> *Given an array, find the average of all contiguous subarrays of size ‘K’ in it.*
>
> |  |   |
> | :--------: | :---: |
> | **INPUT**  | arr = [1, 3, 2, 6, -1, 4, 1, 8, 2], K = 5 |
> | **OUTPUT**   | [2.2, 2.8, 2.4, 3.6, 2.8] |

| No  | Algorithm | Time Complexity |
| :-: |:--------: | :---: |
| 01  | Nested Loop | N * K |
| 02  | Sliding Window | N |
